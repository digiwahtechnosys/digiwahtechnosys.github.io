Out of Scope

What's Not Included

The following items fall outside this proposal. Any requirements in these areas will be scoped and quoted separately.

🤖

AI / LLM Integrations

Any integration with AI platforms, large language models (e.g. OpenAI, Gemini, Claude), or AI-generated content features are not included in this proposal. API keys and associated costs for such platforms are also outside scope.

🎙️

Voice AI & Conversational Agents

Voice AI assistants, IVR systems, conversational bots, or any voice-based automation tools are not part of this engagement.

🔌

Third-Party Tool Integrations (Unlisted)

Integrations with any platform, software, or API not explicitly mentioned in this proposal — including ERP systems, payroll tools, marketing platforms, or custom databases — are out of scope.

🗺️

Google Maps / GPS API Costs (Business 2)

Live GPS tracking and Google Maps navigation features rely on the Google Maps Platform API. Usage costs are billed directly by Google to the client — not included in the Digiwah setup fee.

💳

Existing Platform Plan Limitations & API Upgrade Costs

This proposal assumes your existing platforms (Unleashed, Xero, etc.) support API access under your current subscription plan. If your existing plan does not include API access, or if API usage requires a plan upgrade, those upgrade or additional API costs are the client's responsibility and are not included in the Digiwah setup fee.

Change Request Policy: Any requirement that falls outside the scope defined in this proposal will be assessed and quoted separately before work begins. We will never start out-of-scope work without written agreement on the additional cost.
